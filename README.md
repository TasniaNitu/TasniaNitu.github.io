# Kazi Tasnia Nitu Portfolio Website

One-page portfolio for Machine Learning Engineer / AI Engineer applications.

## Sections

- Name and headline
- Three-line about section
- Four project cards
- Skills
- Contact links

## Before Publishing

Replace these placeholders in `index.html`:

- Fake News Classifier live link
- AI PDF Chatbot Pro live link
- Personal Finance AI Assistant live link
- Project GitHub links with exact repository URLs

## Deploy On GitHub Pages

1. Create a GitHub repository named `TasniaNitu.github.io`.
2. Upload `index.html`, `styles.css`, and this `README.md`.
3. Go to repository `Settings` > `Pages`.
4. Select branch `main` and folder `/root`.
5. Open `https://TasniaNitu.github.io` after GitHub finishes publishing.

## Deploy On Vercel

1. Push this folder to any GitHub repository.
2. Import the repository in Vercel.
3. Keep the framework preset as `Other`.
4. Deploy.

## Best Portfolio Rule

Every project card should have a working live demo. If a project is not deployed yet, deploy it before sending the portfolio to recruiters.
